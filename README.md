# Restaurant Website (Static)

## Lokal starten
Einfach `index.html` im Browser öffnen oder in VS Code "Live Server" nutzen.

## Deployment (GitHub Pages)
1. Repo erstellen (z. B. `taverna-neo`)
2. Dateien pushen
3. GitHub → Settings → Pages
4. Source: Deploy from a branch → `main` / root
5. URL öffnen

## Anpassen
- Name, Öffnungszeiten, Adresse: `index.html`
- Farben: `styles.css` -> `--accent`, `--accent2`
- Hero Bild: `assets/hero.jpg`
- Karte: In `index.html` im `<iframe>` die URL ersetzen (Google Maps Embed oder OpenStreetMap)
